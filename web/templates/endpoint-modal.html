添加端点
Model Rewrite中有不兼容的设置
留空endpoint
选择端点的API兼容类型
OpenAI兼容端点的API路径
用于刷新OAuth token的端点地址
说明：
• 支持 Anthropic、OpenAI 及其他 OAuth 提供商
• API请求将发送到端点URL地址
• Token URL：指定OAuth token刷新的端点地址（必填）
• 过期时间戳为毫秒级Unix时间戳
• 启用自动刷新时，系统会在token过期前自动刷新
代理配置
配置HTTP或SOCKS5代理以访问上游端点
格式: 主机:端口
说明：
• 留空用户名和密码表示无需认证
• HTTP代理支持 CONNECT 方法用于HTTPS
• SOCKS5代理支持TCP连接代理
模型重写配置
配置规则将匹配的Claude模型重写为其他模型
预设模型模式：
• Haiku系列: claude-*haiku*
• Sonnet系列: claude-*sonnet*
• Opus系列: claude-*opus*
• 所有Claude: claude-*
Max Tokens 覆盖配置
强制覆盖请求中的 max_tokens 参数为指定值
范围: 1-200000
说明：
• 启用后，所有通过此端点的请求的 max_tokens 参数都会被强制设置为指定值
• 设置为 0 或禁用此功能则不会修改请求中的 max_tokens
• 该修改在模型重写后、格式转换前进行
