### 端点编辑对话框
#### Tab标签
- 基本配置

#### 基本配置字段
- 名称 (必填)
- 启用
- 启用此端点
- URL (必填)
- 标签 (Tags)
- 用逗号分隔
- 留空则为万能endpoint
- 端点类型 (必填)
  - Anthropic (Claude)
  - OpenAI Compatible
- 选择端点的API兼容类型
- 路径前缀 (必填)
- OpenAI兼容端点的API路径
- 认证方式 (必填)
  - API Key (x-api-key 请求头)
  - Auth Token (Authorization Bearer)
- 认证值 (必填)
- 输入您的 API Key 或 Token

#### 高级配置
##### 代理配置
- 代理配置
- 启用代理
- 配置HTTP或SOCKS5代理以访问上游端点
- 代理类型
  - HTTP
  - SOCKS5
- 代理地址
- 格式: 主机:端口
- 用户名 (可选)
- 代理认证用户名
- 密码 (可选)
- 代理认证密码
- 说明：
  - 留空用户名和密码表示无需认证
  - HTTP代理支持 CONNECT 方法用于HTTPS
  - SOCKS5代理支持TCP连接代理

##### 模型重写配置
- 模型重写配置
- 启用模型重写
- 配置规则将匹配的Claude模型重写为其他模型
- 添加规则
- 选择预设模型
  - Haiku 系列
  - Sonnet 系列
  - Opus 系列
  - 所有 Claude
  - 自定义通配符
- 通配符模式
- 目标模型 (如: deepseek-chat)
- 推荐: deepseek-chat
- 预设模型模式：
  - Haiku系列: claude-*haiku*
  - Sonnet系列: claude-*sonnet*
  - Opus系列: claude-*opus*
  - 所有Claude: claude-*

## 7. 通用术语统一

### 状态相关
- 活跃/激活 → 统一使用"活跃" (Active)
- 不可用/非活跃 → 统一使用"不可用" (Inactive)
- 启用/开启 → 统一使用"启用" (Enabled)
- 禁用/关闭 → 统一使用"禁用" (Disabled)

### 操作相关
- 保存 (Save)
- 取消 (Cancel)
- 编辑 (Edit)
- 删除 (Delete)
- 复制 (Copy)
- 添加 (Add)
- 重置 (Reset)
- 刷新 (Refresh)

### 技术术语
- 端点 (Endpoint) - 统一使用，不使用"终端"
- 请求 (Request)
- 响应 (Response)
- 代理 (Proxy)
- 认证 (Authentication)
- 配置 (Configuration)
- 日志 (Log)
- 标签 (Tag)
- 优先级 (Priority)
- 超时 (Timeout)
- 成功率 (Success Rate)

### 数据相关
- 请求体 (Request Body)
- 响应体 (Response Body)
- 请求头 (Request Headers)
- 响应头 (Response Headers)
- 请求大小 (Request Size)
- 响应大小 (Response Size)

### 提示信息
- 必填字段标记：<span class="text-danger">*</span>
- 可选字段标记：(可选)
- 字段说明使用 <small class="text-muted"> 样式

## 10. 备注

1. 所有中文文本都应该保持简洁明了
2. 技术术语保持一致性，避免同义词混用
3. 用户界面文字应该友好易懂
4. 错误信息应该清晰指出问题所在
5. 确认对话框应该让用户明确知道操作后果
6. 状态显示应该使用统一的词汇和颜色编码
